Project Description

This was one of my first web design classes where I learned how to design a mockup website from scratch. The website was for a zoo called horizon zoo and they needed a website to showcase what they were all about. I not only designed the website in Figma I also coded every page on the website. My biggest struggle was making the design of the website and make it look decent. Designing websites is not my strong suit but I am slowly getting better the more I do it. The more I progress in my web and user experience degree I have a preference to just have someone else make the design and I just code the website because I enjoy coding more than designing.

Bryce Cottam

Full Stack Web Design Student at Weber State University
